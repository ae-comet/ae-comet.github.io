## How to set up and deploy

1. Clone from github

        $ git clone git@github.com:hiyoko3/hiyoko3.github.io.git
        
2. Install packages

        $ npm i
        
3. Develop an app

        $ gulp
        
    If browser start up, we can edit any codes.
    
4. Deploy

        $ git add .
        $ git commit
        $ git push origin master
